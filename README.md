- `pandas` (Version 0.22.0)
- `sklearn` (Version 0.18.1)
- `tensorflow`(Version 1.0.0)
- `keras` (Version 2.0.3)
- `ipywidgets` (Version 6.0.0)
- `bqplot` (Version 0.9.1)
